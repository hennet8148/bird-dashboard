How It Works

The Bird Monitor system is built on a Raspberry Pi 4 running 24/7 with a low-noise omnidirectional microphone. Every 30 seconds, it records a 30-second .wav file. These recordings are analyzed in batches using BirdNET, with a confidence threshold set at 0.1 and sensitivity at 1.0.

We intentionally set the confidence threshold low to capture marginal detections — faint calls, distant songs, and background species — allowing for richer long-term queries and exploratory analysis. While this does introduce more uncertainty, we believe it’s worth it for building a more complete acoustic picture of life on the farm.

Identified species and metadata are written to a MySQL database hosted on the same web server that runs this dashboard. The dashboard itself is built with PHP and JavaScript to display real-time stats and species activity.
